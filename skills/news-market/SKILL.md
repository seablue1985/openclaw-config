# News Market - 全球股市早间要闻

## Overview

生成每日全球股市早间要闻报告，包含：
- 全球股市相关新闻摘要
- A股大盘体温（涨跌停、成交量、情绪指标）
- 板块涨跌热力图
- 看涨/看跌判断及操作建议

## 使用方式

### 手动触发
用户发送以下指令时执行：
- "生成早间要闻"
- "今日股市消息"
- "全球股市早餐"

### 定时任务
每天 08:50 自动执行（需配置 cron）

## 数据来源

1. **A股数据**：AkShare / 腾讯财经
2. **港股数据**：AkShare
3. **美股数据**：yfinance
4. **新闻资讯**：东方财富网、同花顺

## 输出格式

生成结构化报告，包含：
1. 📈 **全球市场概览** - 各主要市场涨跌幅
2. 🇨🇳 **A股大盘体温** - 涨跌停数量、成交量、情绪评分
3. 🔥 **板块热力图** - 涨幅前5/跌幅前5
4. 🎯 **判断与建议** - 看涨/看跌、仓位建议

## 脚本执行

```bash
# 生成早间报告
python3 scripts/generate_morning_news.py

# 指定输出路径
python3 scripts/generate_morning_news.py --output /path/to/report.md
```

## 配置项 (scripts/config.py)

- `REPORT_TIME`: 报告生成时间
- `MARKETS`: 关注的市场列表
- `SECTOR_COUNT`: 板块显示数量
- `FEISHU_WEBHOOK`: 飞书推送地址（可选）
